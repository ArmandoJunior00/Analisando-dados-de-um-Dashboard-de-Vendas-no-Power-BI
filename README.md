# 📊 Projeto de Análise de Vendas com Power BI

## 📌 Descrição do Projeto
Este projeto apresenta um dashboard interativo desenvolvido no **Power BI** para análise de dados de vendas de diferentes produtos e categorias. O objetivo é transformar dados brutos em informações visuais e estratégicas, facilitando a compreensão do desempenho das vendas e auxiliando na tomada de decisões baseada em dados.

O dashboard permite explorar métricas importantes, identificar tendências e analisar o comportamento das vendas ao longo do tempo.

---

## 🎯 Objetivos

- Analisar o desempenho de vendas por **produto, categoria e período**
- Identificar **produtos mais vendidos**
- Avaliar o **faturamento total**
- Visualizar **tendências de vendas ao longo do tempo**
- Criar um **dashboard interativo e intuitivo**
